# TRAVIS HP / 公開用まとめ

TRAVISのメインLP＋詳細手順2ページ構成です。

## 公開するファイル

GitHub Pagesで公開する場合は、`site` フォルダの中身をリポジトリ直下へアップロードしてください。

- `index.html`：メインLP
- `register.html`：テレボート登録の詳しい図解ページ
- `vote.html`：入金・レース選択・舟券購入・結果確認の詳しい図解ページ
- `style.css`：全体デザイン
- `script.js`：画像スライダー、拡大表示
- `favicon.svg`
- `assets/`：使用画像一式

## 公開後に使うページURL例

リポジトリが `travis-boat` の場合：

- メインLP：`https://t-ra-vis.github.io/travis-boat/`
- 登録の概要：`https://t-ra-vis.github.io/travis-boat/register.html`
- 投票を知る：`https://t-ra-vis.github.io/travis-boat/vote.html`
- 勝ち筋を見る：`https://t-ra-vis.github.io/travis-boat/#winning`
- やり方を見る：`https://t-ra-vis.github.io/travis-boat/#how`

## LINEリッチメニューに入れるなら

- 登録の概要 → `register.html`
- 投票を知る → `vote.html`
- 勝ち筋を見る → `#winning`
- 俺に聞く / 勝ちに行く → LINEリンク

## 注意

- `site` フォルダごとではなく、`site` の中身をアップロードしてください。
- `index.html` がリポジトリ直下にないと、GitHub Pagesでトップページとして表示されません。
- 公開後、反映まで数分かかる場合があります。
